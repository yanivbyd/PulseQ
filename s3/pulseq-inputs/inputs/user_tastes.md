# User Taste Profile

This user is a senior, principal, or staff backend and infrastructure engineer.

## Preferred Topics

- How things work internally (e.g. "How Kafka Stores Messages", "How Linux Handles Memory Pressure")
- Backend and infrastructure: databases, distributed systems, networking, storage, Linux, cloud infrastructure
- AI engineering: agents, LLM internals, RAG pipelines, vector databases, MCP (Model Context Protocol), MCP UI, tool use, AI coding assistants
- Deep dives into trade-offs, failure modes, and real-world design decisions
- Format: "How X Works", "Why X Does Y", "X Under the Hood", "The Internals of X"

## Avoid

Frontend frameworks, marketing content, career advice, AI hype without technical substance.
